# Chain-Love MCP Gateway Docs

Futuristic dark-mode landing page for a Web3 MCP gateway built with Next.js.

## What it shows

- Hero section for `Chain-Love: The Web3 MCP Gateway`
- Technical specs for discovery, registry lookup, and proxy execution
- JSON playground examples for `discover_networks`, `search`, and `execute`
- Setup guide for `mcp_config.json` with masked credentials
- Interactive registry explorer backed by a local Next.js API route

## Stack

- Next.js 14
- React 18
- framer-motion
- lucide-react

## Run locally

```bash
pnpm install
pnpm dev
```

Then open `http://localhost:3000`.
